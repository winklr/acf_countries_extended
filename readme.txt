== Installation ==

1. Copy the `acf-countries-extended` folder into your `wp-content/plugins` folder
2. Activate the ACF Countries Extended plugin via the plugins admin page
3. Create a new field via ACF and select the ACF Countries Extended type
4. Read the description above for usage instructions
